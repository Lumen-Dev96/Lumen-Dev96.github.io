Homestead

  • 无需部署开发环境

    Laravel Homestead 是一个官方预封装的 Vagrant box，它为你提供了一个完美的开发环境，你无需在本地安装 PHP ，web 服务器，或其他服务软件。并且不用担心系统被搞乱！ Vagrant box 是完全一次性的。如果有什么地方出错了，你也可以在几分钟内销毁并重建 box ！

  • 跨平台

    Homestead 可以运行在 Windows 、 Mac 或 Linux 系统上，并且里面包含了 Nginx Web 服务器、 PHP 7.1 、 MySQL 、 Postgres、 Redis 、 Memcached 、 Node 、 以及所有利于你开发 laravel 应用的其他程序。

  4. 如果出现图中错误，你就得考虑你是不是把 .homestead 文件夹和 Homestead 文件夹搞混了，解决办法就是，进入 ~/.homestead 目录，使用 vagrant destroy 命令销毁 homestead，然后进入 ~/Code/Homestead 目录使用 vagrant up命令即可

    其原理就是，.homestead 目录一般是用于配置我们的 Homestead 的，比如 Homestead.yaml，after.sh，aliases 等文件在这里配置，而 vagrant 命令一般在 Homestead 目录下使用，不要搞错了哦

    对了，Homestead2.0 版本已经可以取消 .homestead 文件夹了，原来的文件转移到 ~/Code/Homestead 里面了，可以把 .homestead 文件夹干掉了，以避免搞混
